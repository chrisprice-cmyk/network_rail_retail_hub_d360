## Technology Stack

- **Salesforce API Version**: Always use the API version found in the `cumulusci.yml` file for all commands (current default is 64.0)
- **Framework**: CumulusCI and QX for deployment automation. Salesforce CLI for org connections.
- **Components**: Lightning Web Components (LWC), Aura, Wave Analytics
- **Testing**: Jest for LWC testing, Playwright for E2E testing
- **Code Quality**: ESLint, Prettier, Husky for git hooks
